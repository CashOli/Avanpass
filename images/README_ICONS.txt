ICÔNES PWA MANQUANTES
======================

Pour que l'application PWA fonctionne correctement, vous devez ajouter deux images :

1. icon-192.png (192x192 pixels)
2. icon-512.png (512x512 pixels)

Ces icônes doivent représenter le logo AvanPass.

OPTIONS :

Option 1 - Créer avec un outil en ligne :
- Visitez https://www.favicon-generator.org/
- Uploadez le fichier logo.svg
- Téléchargez les tailles 192x192 et 512x512
- Placez-les dans ce dossier

Option 2 - Utiliser une couleur unie temporaire :
- Créez une image carrée avec fond violet (#6366f1)
- Ajoutez la lettre "A" en blanc au centre
- Sauvegardez aux tailles requises

Option 3 - Utiliser l'icône SVG temporairement :
- Le logo.svg est déjà présent
- Les navigateurs modernes l'acceptent en attendant les PNG

En attendant, l'application fonctionnera mais l'installation PWA sera limitée.
